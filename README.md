# Hexapod
Hexapod is my first robot that is completely build from scratch.

It is a Learning project where i explore different domains such as : 3D CAD MODELLING, INVERSE KINEMATICS

COMPONENTS USED :
1. SG90 servo motor X18
2. PCA9685 PWM based servo driver
3. Arduino Nano
4. 2S Li-ion battery / External Wired power source.
